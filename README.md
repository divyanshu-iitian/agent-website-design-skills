# Agent Website Design Skills

Open-source Codex skills for agents that design, implement, audit, and polish websites.

## Skills

- `web-visual-direction`: define a coherent visual concept before building.
- `web-design-system`: create practical tokens, components, states, and layout rules.
- `landing-page-craft`: build landing pages that communicate and convert.
- `responsive-ui-qa`: test and fix layouts across desktop and mobile.
- `conversion-accessibility-polish`: improve clarity, trust, forms, CTAs, and accessibility.

## Use

Copy any skill folder into your Codex skills directory, or reference the folder directly when working with an agent that supports local skills.

Each skill is self-contained with:

- `SKILL.md`
- `agents/openai.yaml`
- optional `references/`
